## 換電腦使用方式

在另一台電腦開 PowerShell：

```powershell
git clone https://github.com/kenTseng0428/test.git
cd test
.\serve-static.ps1 -Port 8001
```

然後用瀏覽器打開：

```text
http://localhost:8001/
```

如果 `8001` 被占用，可以改用：

```powershell
.\serve-static.ps1 -Port 8000
```

並打開：

```text
http://localhost:8000/
```
